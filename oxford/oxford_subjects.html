📚 General Studies Paper 1 (GS1)

This section is shared by both Aryan and Shital. It covers History, Geography, Society, and Art & Culture in a strategic order.

🗿 1. Art & Culture

  • 📘 Truthbooks:
    • NCERT Class 11 Fine Arts
    • Nitin Singhania (Selective + Notes Making)
    • CCRT Official Website (for extra notes)
  • 🧠 Study Order:
    1. Start with NCERT + CCRT to build base
    2. Layer with Nitin Singhania selectively (use only important chapters)
    3. Make one final digital note document

🏰 2. Ancient & Medieval History

  • 📘 Truthbooks:
    • NCERT Class 11 Ancient & Medieval
    • RS Sharma (Old NCERT for Ancient)
    • Satish Chandra (Selective Medieval)
    • Vision IAS Summary PDFs
  • 🧠 Study Order:
    1. Complete RS Sharma + Class 11 Ancient
    2. Then selective Medieval from Satish Chandra
    3. Revise both with Vision IAS summaries

🇮🇳 3. Modern History

  • 📘 Truthbooks:
    • NCERT Class 12 Modern History
    • 📕 Spectrum by Rajiv Ahir (Bible for Pre + Mains)
    • India’s Struggle for Independence – Bipan Chandra (optional deep dive)
  • 🧠 Study Order:
    1. Read NCERT Class 12 first (timeline building)
    2. Then read Spectrum line-by-line (make detailed notes)
    3. Optional: Read Bipan Chandra for understanding ideology & narrative

🌍 4. World History

  • 📘 Truthbooks:
    • Norman Lowe – Mastering Modern World History
    • Vision IAS World History Notes (Optional Condensed)
  • 🧠 Study Order:
    1. Start with Norman Lowe (chapter-wise)
    2. Then compress into 1-page notes per chapter
    3. Use Vision IAS notes for revision

🧭 5. Indian Society

  • 📘 Truthbooks:
    • NCERT Class 11 & 12 Sociology (Indian Society + Social Change)
    • Vision Mains 365 (Society Section)
  • 🧠 Study Order:
    1. Read both NCERTs (make concise notes)
    2. Link with current issues via Vision Mains 365

🗺️ 6. Geography

  • 📘 Truthbooks:
    • NCERT Class 6–12 Geography
    • GC Leong – Physical Geography
    • Certificate Physical & Human Geography (Goh Cheng Leong)
    • PMFIAS Notes (Advanced)
  • 🧠 Study Order:
    1. Complete NCERTs (6–10 quick skim, 11–12 in detail)
    2. GC Leong + PMFIAS for depth
    3. Revise using diagrams, flowcharts, and notes
🧱 Phase 1: Foundation - Core Subjects
🧠 GS1 - Indian Heritage & Culture, History, Geography
  • Ancient History: RS Sharma (Old NCERT), Tamil Nadu Class 11-12
  • Medieval History: Satish Chandra (Old NCERT)
  • Modern History: Spectrum Rajiv Ahir + Bipin Chandra (Selective)
  • Art & Culture: Nitin Singhania, CCRT, NCERT Class 11 Fine Arts
  • Geography: NCERT Class 6–12 + GC Leong + PMF IAS Notes
⚖️ GS2 - Polity, Governance, IR
  • Polity: Laxmikanth + Constitution Bare Act + DD Basu (Selective)
  • Governance: ARC Reports + 2nd ARC + PIB + Newspaper
  • International Relations: MEA website, Current Affairs, India Yearbook
📈 GS3 - Economy, Environment, Security, Tech
  • Economy: Sanjeev Verma, Sriram IAS Notes, NCERTs, Budget + Survey
  • Environment: Shankar IAS Book + Current Affairs + NCERTs
  • Disaster Management: NDMA Guidelines, Yojana/Kurukshetra
  • Science & Tech: Current Affairs, PIB, Prelims-focused Notes
  • Internal Security: Lexicon, Current Affairs
🧭 GS4 - Ethics, Integrity & Aptitude
  • Theory: Lexicon, Subbarao, G. Subba Rao + 2nd ARC Ethics Report
  • Case Studies: Vision/Forum compilations, Previous Year Papers
  • Real Life Examples: Daily Current Affairs + Notes Making
📝 Essay Paper
  • Reading: Yojana, Kurukshetra, The Hindu Editorials
  • Practice: Weekly Essays, Peer Reviews, Quote Banks
🔢 CSAT
  • Books: Arihant CSAT, Previous Year Papers
  • Topics: Comprehension, Logical Reasoning, Data Interpretation, Maths (Basic)
🔥 Phase 2: Optional Subject + Current Affairs Mastery
📰 Current Affairs Strategy
  • Newspaper: The Hindu / Indian Express (1 hour daily)
  • Magazines: Yojana, Kurukshetra, EPW (Selective Reading)
  • Compilations: Vision IAS / Insights Monthly
  • Daily Revision: Maintain self-made current affairs notes
  • Prelims Focus: PIB, All India Radio, India Yearbook
  • Mains Focus: Issues + Opinions + Examples from editorials
📚 Optional Subject Selection
  • Criteria to Choose: Background Knowledge, Interest, Overlap with GS, Availability of Material
  • Popular Optionals: PSIR, Anthropology, Sociology, Geography, Philosophy, History, Literature
  • Resources: Standard Books + Toppers’ Notes + Coaching PDFs
  • Test Series: Join after completing 60–70% syllabus
  • Answer Writing: Practice Optional + GS answer writing together from Phase 3
🖋️ Phase 3: Answer Writing + Prelims & Mains Test Series
✍️ Answer Writing Strategy
  • Start After: Basic NCERT + Standard Book coverage
  • Platform: Insights Secure / ForumIAS / Self Practice
  • Method: Write 1–2 GS answers daily (Use intro-body-conclusion model)
  • Review: Use peer reviews or toppers’ copies for self-check
  • Structure Focus: Bullet points, headings, underlining, examples, diagrams
  • Timing: Practice writing within strict 7–11 minute window per answer
📑 Prelims Test Series Strategy
  • Start: 6–8 months before Prelims
  • Sources: Vision IAS, Insights, Vajiram, ClearIAS
  • Schedule: Weekly / Bi-weekly mocks based on syllabus portion
  • Analysis: Don’t just attempt — analyse every mistake
  • Target: 35+ full-length mocks before Prelims
  • PYQ Practice: UPSC Previous Year Papers from 2011–present
📄 Mains Test Series Strategy
  • Start: After completion of 70–80% syllabus
  • Sources: Vision IAS Mains / ForumIAS MGP / GS Score
  • Tests: Optional + GS Mains sectional and full-length
  • Self Evaluation: Improve intro, value addition, structure
  • Peer Evaluation: Exchange answers with mentors / friends
🔁 Phase 4: Final Revision & Mastery Loop
📚 Full Syllabus Revision System
  • Revision Cycles: Use 3–2–1 Rule (3 revisions before Prelims, 2 before Mains, 1 before Interview)
  • Schedule: Reverse planning based on exam date
  • Tools: OneNote / Notion / Handwritten summary books
  • Coverage: Static + Dynamic + Optional
  • Mindmaps: Make GS2/GS3/Essay mindmaps for value-packed revision
🎯 Prelims–Mains Balance System
  • 90 Days Before Prelims: Switch to full Prelims focus
  • After Prelims: Immediately switch back to Mains-focused answer writing
  • Optional Backup: Keep 20% optional touch during prelims phase
  • Essay Writing: Weekly essay writing till Prelims, then twice a week post-Prelims
🧠 Mastery Techniques
  • Flashcards: Use Anki / Physical flashcards for facts, Acts, Article numbers
  • Peer Quizzes: Weekly sessions with friend or AI-based quiz tools
  • Recall Sheets: A4 sheets to test knowledge without seeing books
  • Output Focus: Use Feynman Technique to explain topics to others
  • Active Recall: Daily active recall cycles using keywords + flowcharts
🧑‍⚖️ Phase 5: UPSC Interview & Persona Engineering
📝 DAF Mastery (Detailed Application Form)
  • Deep Analysis: Line-by-line breakdown of your DAF
  • DAF Book: Prepare a mini-book with 20+ questions for each DAF detail (hometown, hobby, etc.)
  • DAF-Based Questions: Personality traits, life choices, goals — all from your form
🎙️ Interview Simulation & Practice
  • Mock Interviews: At least 5–7 with expert panels
  • Record Yourself: Video analysis of posture, tone, and expressions
  • Situational Questions: Ethical dilemmas, administration cases, national issues
  • Voice & Diction: Speak calmly, slow, with confidence (record daily)
  • Panel Dynamics: Handling multiple panelists — smile, eye contact, logic
🧠 Persona Engineering

  • Inner Compass: Clarity of values, beliefs, vision for the country
  • UPSC Fit: What kind of officer UPSC is looking for? Reflect that in answers
